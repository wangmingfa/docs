# 常用软件

## yay

### 安装 git和 base-devel包组​​（用于从 AUR 编译软件）

```bash
sudo pacman -S --needed git base-devel
```

### 克隆 yay 的 AUR 仓库

```bash
git clone https://aur.archlinux.org/yay.git
```

### 编译yay

```bash
cd yay && makepkg -si
```

### 验证yay安装

```bash
yay --version
```
